🔧 Mise en place du backend Google Apps Script

Suivez ces étapes une seule fois pour connecter l'application à Google Sheets :

1. Ouvrez Google Sheets et créez un nouveau classeur nommé SSIAP2_Data.

2. Dans le menu Extensions → Apps Script, collez le contenu du fichier Code.gs fourni.

3. Lancez la fonction setup() depuis le menu Exécuter pour initialiser les onglets.

4. Déployez : Déployer → Nouveau déploiement → Application Web. Accès : Tout le monde. Copiez l'URL.

5. Collez cette URL dans le champ de la page de connexion. Elle sera mémorisée dans votre navigateur.

🔐 Changer le mot de passe admin

Modifiez la constante ADMIN_PASSWORD au début du fichier Code.gs, puis redéployez en cliquant sur Déployer → Gérer les déploiements → Modifier → Nouvelle version.
